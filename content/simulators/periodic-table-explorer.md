# Periodic Table Explorer

Explore all 118 elements with our interactive periodic table. Click any element to see detailed information about its properties, uses, and atomic structure.

## Features

- **Interactive Grid**: Click any element for details
- **Color Coding**: Elements grouped by type (metals, nonmetals, etc.)
- **Search**: Find elements by name, symbol, or atomic number
- **Detailed Info**: Atomic mass, electron configuration, and more

## What You Can Explore

### Element Details

For each element, view:

- Atomic number and mass
- Electron configuration
- Group and period
- Melting and boiling points
- Common uses and applications

### Trends and Patterns

- Compare elements across groups
- Observe periodic trends
- Understand element families

## Try These Activities

1. Find all the noble gases (Group 18)
2. Compare the alkali metals (Group 1)
3. Explore the transition metals
4. Find elements that are gases at room temperature
